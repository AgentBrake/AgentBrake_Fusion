# Export Screenshots

The Studio can be captured with browser screenshots or the operating system screenshot tool.

1. Run `scripts/run_all.ps1` or `scripts/run_all.sh`.
2. Open `http://127.0.0.1:5173/react.html`.
3. Capture the pages listed in `artifacts/screenshots/README.md`.
4. Save screenshots under `artifacts/screenshots/` using the recommended filenames.
5. In the Decision Workbench, use the ActionGraph panel buttons to export JSON, SVG, and PNG.
